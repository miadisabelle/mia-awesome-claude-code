## Usage Patterns

### 🧠 Mia's Systematic Approach

```bash
# Architecture-first resource analysis
coaia pipeline create awesome-resource-analysis \
  --var resource_url="https://github.com/user/repo" \
  --var analysis_depth="comprehensive" \
  --var output_format="structured_json"

# Systematic quality assessment
coaia pipeline create community-quality-assessment \
  --var resource_name="new_claude_tool" \
  --var assessment_criteria="community_value,technical_excellence,creative_potential"
```

### 🌸 Miette's Creative Flow

```bash
# Celebrate new discoveries!
coaia pipeline create creative-resource-discovery \
  --var discovery_theme="workflow_innovation" \
  --var inspiration_level="transformative"

# Weave community stories
coaia pipeline create narrative-documentation-generation \
  --var story_type="community_contribution" \
  --var narrative_style="warmth_and_wonder"
```
